A prediction model for olympic medals. This model is based on the data of the olympic games from 1896 to 2016.

The model is trained on the data of the olympic games from 1896 to 2012 and tested on the data of the olympic games from 2012 to 2016.

The model is a simple linear regression model using scikit-learn that predicts the number of medals that a country will win in the next olympic games based on the number of medals that the country has won in the previous olympic games.
